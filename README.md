# Django Learning Project

## Overview

This is a beginner Django project created to learn the basics of the Django framework. The project includes simple pages like a Home page and an About page that provides information about Mars.

## Features
	•	Home page using Django templates
	•	About page (About Mars)
	•	Basic URL routing
	•	Simple Django views

## Technologies
	•	Python
	•	Django
	•	HTML
## Setup
	1.	Clone the repository:
	2.	Install Django:
      pip install django
	3.	Run the server:  
      python manage.py runserver
  	4.	Open in browser:
      http://127.0.0.1:8000/
## Purpose

This project was built for learning Django fundamentals like templates, views, and routing.
